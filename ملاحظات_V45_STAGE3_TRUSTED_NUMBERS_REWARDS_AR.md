# ملاحظات V45 Stage 3 - مكافآت الأرقام الموثوقة / نقاط البيع

تمت إضافة خيار مستقل لكل رقم موثوق / نقطة بيع للتحكم في المكافآت:

- إضافة حقل rewardsEnabled داخل بيانات الرقم الموثوق.
- عند إضافة أو تعديل رقم موثوق يظهر خيار: تفعيل المكافآت لهذا الرقم.
- في قائمة الأرقام الموثوقة يظهر هل المكافآت مفعلة أو معطلة لهذا الرقم.
- عند وصول طلب من رقم موثوق يتم تنفيذ البيع كالمعتاد.
- إذا كانت مكافآت الرقم مفعلة وكان نظام المكافآت العام مفعلاً: يتم احتساب النقاط والمكافآت.
- إذا كانت مكافآت الرقم معطلة أو النظام العام للمكافآت متوقفاً: يتم البيع بدون احتساب نقاط جديدة.

لم يتم تعديل:
- مخزون الكروت.
- النسخ والاسترجاع.
- المحافظ.
- الواتساب.
- Gradle أو GitHub Actions.
- الترخيص.
