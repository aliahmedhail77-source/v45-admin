# ONLINE Admin Stable V37

نسخة مستقرة مضافة إليها تحسينات نظام المكافآت ومركز التحديثات.

## أهم الإضافات
- مخزون مستقل لكروت المكافآت، منفصل عن مخزون البيع العادي.
- كروت المكافأة لا تضيف نقاطًا ولا تدخل في أرباح البيع.
- إدارة النقاط والمكافآت بزر ذهبي مميز ومتحرك.
- نصوص رسائل النقاط والمكافآت قابلة للتعديل بحد 60 حرفًا لكل نص.
- مركز تحديثات داخل التطبيق: فحص تحديث + عرض مميزات + فتح رابط APK.
- حفظ نظام الترخيص الحالي والتحديث فوق النسخ السابقة بدون حذف إذا كان APK Release وبنفس التوقيع.

## التثبيت
ارفع محتويات هذا المجلد إلى GitHub ثم شغل Actions، وثبت فقط app-release.apk.
