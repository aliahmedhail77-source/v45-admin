# ملاحظات V45 Stage 4 - صلاحية النقاط وكشوفات واتساب

## المضاف
- إضافة قسم جديد داخل مكافآت العملاء باسم: الصلاحية.
- إضافة خيار: تصفير النقاط عند عدم التعبئة.
- مدة عدم النشاط افتراضياً: 10 أيام، ويمكن تعديلها.
- عند تفعيل الخاصية: إذا لم يقم العميل بأي تعبئة خلال المدة المحددة، يتم تصفير نقاطه وتسجيل عملية مصادرة في السجل.
- قبل المصادرة: يعرض النظام العملاء الذين بقي لهم يومان أو أقل على انتهاء النقاط.
- لكل عميل قرب انتهاء النقاط: زر عرض كشف + زر إرسال واتساب أعمال.
- سجل النقاط المصادرة: يعرض العملاء الذين تمت مصادرة نقاطهم مع التفاصيل.
- لكل عملية مصادرة: زر عرض كشف المصادرة + زر إرسال واتساب أعمال.
- عند تعبئة عميل بعد انتهاء نقاطه، تتم مصادرة النقاط القديمة أولاً ثم تحتسب نقاط التعبئة الجديدة.

## الحماية
- الخاصية لا تعمل إلا إذا تم تفعيلها من قسم الصلاحية.
- لا يوجد إرسال واتساب تلقائي؛ الإرسال يدوي بعد المراجعة.
- لا يتم تعديل مخزون الكروت أو الواتساب أو المحافظ أو الترخيص أو Gradle.
