# ملاحظات V45 Stage 2 - Rewards Control

تمت إضافة التحكم بالمكافآت بدون تعديل النسخ الاحتياطي أو الكروت أو الواتساب أو Gradle أو الترخيص.

## البيع المباشر
- إضافة خيار داخل شاشة البيع المباشر: احتساب المكافأة لهذه العملية.
- عند تفعيل الخيار يتم احتساب نقاط العميل وإرفاق رسالة النقاط/المكافأة مع رسالة الكرت.
- عند إغلاق الخيار يتم البيع طبيعيًا بدون احتساب نقاط جديدة.
- إذا كان نظام المكافآت العام متوقفًا من الإعدادات، يظهر تنبيه ولا يتم احتساب المكافأة.

## نقاط البيع
- إضافة خيار لكل نقطة بيع: مع تفعيل المكافآت لهذه النقطة / بدون مكافآت.
- يظهر وضع المكافآت في قائمة نقاط البيع.
- يتم حفظ وضع المكافآت داخل بيانات نقطة البيع.
- عند إنشاء دفعة POS يتم تضمين حالة المكافآت داخل ملف الدفعة كحقل rewards_enabled حتى يستفيد منه تطبيق نقطة البيع عند دعمه.
