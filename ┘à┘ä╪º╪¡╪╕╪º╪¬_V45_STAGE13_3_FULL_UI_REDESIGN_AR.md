# V45_STAGE13.3_FULL_UI_REDESIGN

## نوع التحديث
تحديث واجهات فقط. لم يتم تعديل منطق السلف أو السداد أو SMS أو قاعدة البيانات أو PDF.

## ما تم تحسينه
- اعتماد نمط Dark Neon UI على مستوى التطبيق.
- خلفية داكنة موحدة.
- هيدر بتدرج سماوي/أزرق/بنفسجي.
- كروت داكنة بحواف نيون ولمعة خفيفة.
- تأثير ضغط موحد على الكروت والأزرار: تكبير بسيط + رفع الظل.
- تحسين ألوان النصوص حتى لا تظهر باهتة.
- تحويل ثيم التطبيق إلى Theme.Material.NoActionBar داكن لتقليل ظهور النوافذ البيضاء.
- تحسين حقول إضافة/تعديل الزبون والقيد اليومي لتظهر داكنة وواضحة.
- تحسين خيارات طريقة التعامل مع العميل داخل نافذة التعديل بألوان واضحة.

## شاشات شملها التحسين الشكلي
- الرئيسية.
- شريط التنقل السفلي.
- الدفتر المحاسبي.
- إضافة/تعديل زبون.
- إضافة قيد يومي.
- الإعدادات.
- البطاقات والأزرار العامة التي تستخدم cardBox/action.

## لم يتم لمسه
- SmsProcessor.
- PaymentParser.
- AppStore logic.
- منطق السقف والرصيد والسداد.
- تقارير PDF.
- الاستيراد والمخزون.
