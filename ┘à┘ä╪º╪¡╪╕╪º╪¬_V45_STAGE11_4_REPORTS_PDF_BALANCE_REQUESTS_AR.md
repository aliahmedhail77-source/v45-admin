# V45 Stage 11.4 - تقارير PDF + مشاركة واتساب + طلبات الرصيد

تم بناء هذا التحديث فوق النسخة المستقرة V45 Stage 11.3 بدون تعديل قلب البيع أو منطق السحب من المخزون.

## الإضافات

1. إصلاح ظهور تقرير نقطة البيع بخلفية بيضاء ونص غير واضح:
   - أصبح نص تقرير المعاينة واضحاً داخل نافذة التقرير.
   - تمت إضافة خيار مشاركة التقرير كـ PDF.

2. تقارير PDF منظمة:
   - تقرير PDF لكل عملية من السجل.
   - تقرير يومي PDF.
   - تقرير أسبوعي PDF.
   - تقرير شهري PDF.
   - تقرير PDF حسب فترة مخصصة.
   - تقرير PDF خاص بنقطة بيع حسب التاريخ.

3. مشاركة التقارير:
   - زر مشاركة PDF يفتح نافذة المشاركة لاختيار واتساب أو واتساب أعمال أو أي تطبيق آخر.
   - الإرسال يدوي عبر نافذة المشاركة لتجنب إرسال تقارير بالغلط.

4. طلب رصيد من نقطة البيع:
   - إذا أرسلت نقطة البيع رسالة موقعة ببصمتها مثل:
     طلب رصيد
     الشرعبي
   - أو:
     طلب رصيد 5000
     الشرعبي
   - يتم تسجيل طلب رصيد داخل السجل كعملية تحتاج مراجعة.
   - يظهر إشعار: نقطة بيع تحتاج رصيد.
   - لا يتم تعبئة الرصيد تلقائياً، بل يظل القرار بيد المدير من زر تعبئة رصيد.

## لم يتم تغييره

- لم يتم تعديل سحب الكروت من المخزون.
- لم يتم تعديل بصمة نقطة البيع.
- لم يتم تعديل السقف أو خصم الرصيد.
- لم يتم تعديل منع التكرار.
- لم يتم تعديل إرسال الكروت للعميل.
