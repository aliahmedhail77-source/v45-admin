# ملاحظات V45 Stage 7 - حماية الدخول بالبصمة و PIN

## التعديل المنفذ
- إضافة قسم جديد داخل الإعدادات باسم: الأمان والدخول.
- إضافة اختيار طريقة الدخول للمستخدم:
  - بدون حماية.
  - PIN فقط.
  - بصمة فقط.
  - بصمة + PIN احتياطي.
- إضافة شاشة قفل عند فتح التطبيق إذا كانت الحماية مفعلة.
- دعم إدخال PIN من 4 إلى 8 أرقام.
- تشفير PIN داخلياً عبر SHA-256 مرتبط بمعرف الجهاز.
- دعم بصمة الإصبع عبر BiometricPrompt للأجهزة المدعومة.
- إضافة أزرار اختبار للبصمة و PIN داخل إعدادات الأمان.

## ملاحظات مهمة
- لا يتم تفعيل البصمة إذا كان الجهاز لا يدعمها أو لم يتم إعداد بصمة من إعدادات الهاتف.
- الوضع الأفضل للمستخدم هو: بصمة + PIN احتياطي.
- لم يتم لمس البيع، الكروت، المحافظ، المكافآت، الإشعارات، النسخ والاسترجاع، Gradle، أو الترخيص.
