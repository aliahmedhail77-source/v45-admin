# كرت برو - Stage 14.8 Smart Wallet Link Resolver

## مبني من
KrtPro_STAGE14_7_9_AUTO_SMS_STABILITY_GUARD_PROJECT.zip

## الهدف
حل مشكلة المحافظ التي ترسل حوالة باسم فقط أو رقم حساب فقط، وتسريع إضافة الربط مباشرة من بطاقة المراجعة.

## الجديد
1. زر سريع في بطاقة المراجعة:
   🔗 ربط وإرسال سريع

2. عند الضغط:
   - يملأ التطبيق اسم المحفظة تلقائياً.
   - يستخرج الاسم من الرسالة إن وجد.
   - يستخرج رقم الحساب القصير إن وجد.
   - يطلب منك رقم العميل فقط غالباً.
   - زر: حفظ الربط وإرسال الكرت.

3. بعد الحفظ:
   - يحفظ الربط كمصدر موثوق.
   - يسمح بالإرسال الآلي مستقبلًا لنفس الاسم/الحساب.
   - إذا اخترت حفظ الربط وإرسال الكرت، يرسل كرت العملية الحالية فوراً.

## قاعدة الأمان
- لا يعتمد التطبيق على كلمة "مشترك" وحدها.
- الربط يكون على المحفظة + رقم الحساب إن وجد، أو المحفظة + الاسم كما ظهر في الرسالة.
- لا يرسل آلياً إلا بعد حفظ الربط مرة واحدة.
